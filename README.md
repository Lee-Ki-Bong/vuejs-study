#

## 프로젝트 관리 명령어

### Project setup

```
yarn install
npm install

```

### 개발 모드에서 컴파일 및 핫 리로딩

```
yarn serve
npm run serve

```

### 프로덕션용 컴파일 및 최소화

```
yarn build
npm run build
```

### 코드 스타일 검사 및 파일 수정

```
yarn lint
npm run lint
```
